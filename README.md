# Personal Landing Page

My personal landing page — a single-page site built with HTML and CSS to introduce myself as a frontend developer.

## Features
- Hero section — short intro with a CSS-code-styled visual card
- About section — background and work approach
- Skills section — HTML5, CSS3, responsive design
- Projects gallery — showcase of previous work
- Contact section — email, phone, Telegram, and GitHub links
- Fully responsive design (mobile, tablet, desktop)

## Technologies
- HTML5
- CSS3 (Flexbox, Grid, animations)
